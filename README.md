# autossh_init-cookbook

Configure autossh_init

## Supported Platforms

* redhat centos fedora scientific amazon oracle

## Usage

### autossh_init::default

Include `autossh_init` in your node's `run_list`:

```json
{
  "run_list": [
    "recipe[autossh_init::default]"
  ]
}
```
